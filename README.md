# Deal Land Real Estate & Builders

**Website for Deal Land Real Estate & Builders — Bahria Town, Lahore**

## Files

| File | Description |
|------|-------------|
| `index.html` | Main HTML structure |
| `style.css`  | All styles and responsive layout |
| `script.js`  | All JavaScript interactions |

## Features

- Fully responsive (mobile, tablet, desktop)
- Mobile hamburger navigation
- Property listing filter (All / For Sale / For Rent / Plots)
- Inquiry form with validation
- Scroll-based fade animations
- WhatsApp floating button
- Google Maps integration
- SEO meta tags + JSON-LD schema

## How to Deploy on GitHub Pages

1. Upload all 3 files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/repo-name`
